# 📝 Todo App

A modern and responsive **Todo App** built using **HTML, CSS, and JavaScript** in a single file.

## ✨ Features

- Add, edit, and delete tasks  
- Mark tasks as completed  
- Filter tasks (All / Pending / Completed)  
- Add date and time for tasks  
- Local storage support  
- Responsive premium UI  

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript  

## 🚀 Run Locally

Simply download the project and open:

```bash
index.html
```

in your browser.
