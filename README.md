# 📝 Todo App

A clean and interactive Todo App built using **HTML, CSS, and JavaScript**.

This project focuses on building a practical UI with real-world features like date & time selection, task management, and local storage — while keeping the code simple and beginner-friendly.

---

## 🚀 Features

* Add tasks with **date and time**
* Auto-fill current date & time (editable)
* Mark tasks as completed
* Edit tasks inline
* Delete tasks
* Filter tasks (All / Pending / Completed)
* Clear completed tasks
* Live **current date & time display**
* Tasks saved using **localStorage**
* Clean and modern UI

---

## 🛠 Tech Stack

* HTML
* CSS
* JavaScript (Vanilla JS)
